# Creative Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Perfect for showcasing your projects and skills.

## Features

✨ **Modern Design**
- Clean, professional aesthetic
- Beautiful gradient backgrounds
- Smooth animations and transitions
- Fully responsive layout

🎯 **Sections Included**
- **Navigation Bar** - Sticky navigation with smooth scrolling
- **Hero Section** - Eye-catching landing area with call-to-action
- **About Section** - Introduce yourself and list your skills
- **Projects Section** - Showcase 6 featured projects in a grid layout
- **Contact Section** - Contact form for inquiries
- **Footer** - Social links and copyright information

📱 **Responsive Design**
- Mobile-first approach
- Works seamlessly on all devices
- Optimized for tablets, smartphones, and desktops

⚡ **Interactive Features**
- Smooth scroll navigation
- Form validation and submission feedback
- Intersection Observer animations
- Hover effects and transitions
- Mobile-friendly navigation

## Getting Started

1. **Clone or Download** this repository
2. **Open `index.html`** in your web browser
3. **Customize** the content to match your portfolio
4. **Deploy** to your web hosting service

## Customization

### Update Your Information
Edit `index.html` to:
- Change your name and title
- Update the About section with your bio
- Modify the skills list
- Add your actual projects
- Update social media links

### Customize Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* ... other colors ... */
}
```

### Add Real Projects
Replace the project cards with your actual work:
- Update project titles and descriptions
- Add real project images or links
- Change the gradient colors
